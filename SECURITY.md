# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by [opening a GitHub issue](../../issues/new).

Include the following details in your report:

- Description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Suggested fix (if any)

## Response

We will acknowledge your report within 7 days and provide updates as we work toward a resolution.

## Scope

This is a test case library and does not process sensitive data or manage authentication. All reports are welcome, but please keep the project's scope in mind when assessing severity.
